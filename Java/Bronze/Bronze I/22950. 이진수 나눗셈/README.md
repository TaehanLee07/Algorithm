## 문제

이진수 𝑀$M$이 2𝐾$2^{K}$으로 나누어 떨어지는지 여부를 판별하는 프로그램을 작성하시오. 이 때, 나누어 떨어진다는 것은 나머지 없이 정수 몫으로 나누어진다는 것을 의미한다.

## 입력

첫 번째 줄에는 이진수 𝑀$M$의 자리수인 𝑁$N$ (1≤𝑁≤1000000$ 1 \leq N \leq 1\,000\,000$)이 주어진다.

두 번째 줄에는 이진수 𝑀$M$이 𝑁$N$자리만큼 주어진다. 이진수 𝑀$M$의 앞부분에는 불필요한 0이 올 수 있다.

세 번째 줄에는 나누는 수 2𝐾$2^{K}$의 𝐾$K$ (0≤𝐾≤1000000$ 0 \leq K \leq 1\,000\,000$)가 주어진다.

## 출력

이진수 𝑀$M$이 2𝐾$2^{K}$으로 나누어 떨어진다면 `YES`, 나누어 떨어지지 않는다면 `NO`를 출력한다.

## 예제 입력 1

```
8
10000000
6

```

## 예제 출력 1

```
YES

```

## 예제 입력 2

```
5
00110
2

```

## 예제 출력 2

```
NO
```
