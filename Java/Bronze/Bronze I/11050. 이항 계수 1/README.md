## 문제

자연수 𝑁\(N\)과 정수 𝐾\(K\)가 주어졌을 때 이항 계수 (𝑁𝐾)\(\binom{N}{K}\)를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 𝑁\(N\)과 𝐾\(K\)가 주어진다. (1 ≤ 𝑁\(N\) ≤ 10, 0 ≤ 𝐾\(K\) ≤ 𝑁\(N\))

## 출력

(𝑁𝐾)\(\binom{N}{K}\)를 출력한다.

## 예제 입력 1

```
5 2

```

## 예제 출력 1

```
10
```
